Summary: Lattitude vs. Temperature

The results of this graph are quite clear from a simple glance, it seems that the closer one gets to 0 lattitude, generally speaking, the higher the temperature you can expect for this day.

One thing to note is that there are significantly fewer data points at the lowest lattitudes. This is largely because there is no land besides anartica that far south, and what little there is is relativley undeveloped compared to the high lattitudes. While I believe we have enough data to see a clear trend, if I could improve this data set at all, I would grab more data for the lower latitudes.
