Summary: Lattitude vs. Cloud Cover

The cloud cover seems to have absolutely no correlation with lattitude, which makes sense. It also seems to be a very inexact practice, with many clusters around 40% cloud cover, 75% cloud cover, 0 % cloud cover, and other pleasant to human numbers.

These numbers should be taken not as the absolute truth but as a human approximation.
